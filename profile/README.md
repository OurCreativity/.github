## About

OurCreativity is an Indonesian creative community platform connecting 3,000+ young creators across five divisions: Graphics, Coding, Video, Writing, and Meme. This GitHub organization is maintained by the **Coding Division**.

**Mission:** Connect, collaborate, create, and appreciate creative works from talented Indonesian youth.

## Contributing

```bash
# Fork → Clone → Branch → Commit → Push → PR
git clone https://github.com/YOUR_USERNAME/REPO_NAME.git
git checkout -b feature/your-feature
git commit -m "feat: your description"
git push origin feature/your-feature
```

**Commit Types:** `feat` `fix` `docs` `style` `refactor` `test` `chore`

See `CONTRIBUTING.md` in each repository for detailed guidelines.
