# OurCreativity

---

## About

OurCreativity is an Indonesian-born creative community platform designed as a home for young creators across all disciplines. This GitHub organization is maintained by the Coding Division, serving as the technical backbone of the OurCreativity ecosystem.

Founded with the vision of developing the creative spirit of young Indonesians, we connect, collaborate, create, and appreciate works from talented individuals across the nation.

---

## Organization Structure

OurCreativity operates through five specialized divisions:

| Division | Focus Area |
|----------|------------|
| Graphics | Digital art, UI/UX design, illustration |
| Coding | Software engineering, creative coding |
| Video | Film, motion graphics, cinematography |
| Writing | Literature, journalism, poetry |
| Meme | Internet culture, humor, trends |

This GitHub organization represents the **Coding Division**, responsible for all technical development, open-source projects, and platform infrastructure.

## Contributing

We welcome contributions from all developers, especially those aligned with our mission of nurturing young Indonesian creators.

### Getting Started

1. Fork the repository you wish to contribute to
2. Create a feature branch from `main`
3. Commit your changes with clear, descriptive messages
4. Submit a pull request with a detailed description

### Guidelines

- Follow the established code style and conventions
- Write meaningful commit messages
- Include documentation for new features
- Ensure all tests pass before submitting

For detailed contribution guidelines, see [CONTRIBUTING.md](CONTRIBUTING.md) in each repository.
